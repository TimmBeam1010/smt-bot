 # 🤖 SMT Bot (Smart Money Trading Bot)

Автоматический помощник для криптотрейдинга на основе RSI, MACD и волатильности.

## 🚀 Функции
- Анализ рынка (RSI, MACD, волатильность)
- Сигналы в реальном времени
- Личный кабинет с настройками
- Управление рисками (стоп-лосс, тейк-профит)

## 🛠️ Технологии
- Node.js, Express
- Supabase (PostgreSQL)
- Timeweb VPS
- PM2

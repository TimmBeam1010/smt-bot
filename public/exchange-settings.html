📖 Как получить API ключи на бирже

  1. Войдите в свой аккаунт на бирже
  2. Перейдите в раздел API Management или API Keys
  3. Создайте новый API ключ с правами:
  4. Скопируйте API Key и Secret Key
  5. Вставьте их в поля ниже и нажмите Сохранить
🔑 Какие права нужны:
  • Чтение баланса и данных аккаунта
  • Создание и отмена ордеров
  • Просмотр истории сделок
  • ⚠️ НЕ давайте права на вывод средств!
Как получить: документация биржи
Хранится в зашифрованном виде на сервере
🔒 Безопасность: Ваши ключи шифруются на сервере. Мы никогда не передаём их третьим лицам. Рекомендуем привязать IP-адрес вашего сервера к API ключу.
